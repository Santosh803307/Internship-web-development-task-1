# Responsive Landing Page

## 📌 Description
This project is a simple responsive landing page built using HTML and CSS. It follows all the task requirements including header with navigation, hero section with CTA button, features grid, footer with social links, and full responsiveness using media queries.

## 🚀 Features
- ✅ Header with logo and navigation links
- ✅ Hero section with heading, paragraph, and call-to-action button
- ✅ Features section using CSS Grid layout
- ✅ Footer with social media links
- ✅ Responsive design using media queries (navbar collapses on mobile)
- ✅ Flexbox for header and hero layout
- ✅ Hover effects on buttons and links

## 🛠 Tools Used
- HTML5
- CSS3
- VS Code
- Live Server (for testing)
